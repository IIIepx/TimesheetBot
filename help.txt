<b>Справка</b>
*******

<code>/start</code> - регистрация пользователя
<code>/help</code> - вывод справки
<code>/проект</code> - выбор текущего проекта в который будут добавляться данные
<code>/итог</code> - отчет по текущему проекту
<code>/итог общий</code> - отчет по всем проектам

<b>Администратор</b>
*************
<code>/новый</code><i>проект</i> - добавление нового проекта, имя проекта должно быть уникальным
<code>/итог общий</code> - отчет по всем проектам и пользователям


<b>Время вводится в формате целого числа или десятичной дроби </b>
например:
8 или 8.5
В таком виде запись попадет в текущий проект за сегодняшнюю дату

Для добавления записи за другую дату формат команды будет таким:
дд.мм.гг число
Например:
11.01.23 12

В случае ошибочного ввода можно повторить ввод с корректными данными.
Чтобы удалить запись, нужно указать время 0
Например:
11.01.23 0
